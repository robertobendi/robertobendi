#### Selected work

- **[pebblestack](https://github.com/robertobendi/pebblestack)** — A no-bloat PHP + SQLite CMS. WordPress without the marketplace.
- **[MedusaVis](https://github.com/robertobendi/MedusaVis)** — Interactive network graphs for JSON-LD data.
- **[CRATOVITE](https://github.com/robertobendi/CRATOVITE)** — A single bash script that drags Create React App into Vite, kicking and screaming.
- **[WEEP](https://github.com/robertobendi/WEEP)** — A profiling tool for Unity. Born from one too many "why is this frame 80ms" mornings.
- **Neodori Forever** — Arcade racer published by QubicGames.
